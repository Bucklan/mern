# Qorap - E-commerce Website

## Start

1. Step 1:

    ```bash
    git clone https://github.com/bucklan/mern.git
    ```

2. Step 2:

    ```bash
    cd 1st-angular-app
    ```

3. Step 3:

    ```bash
    cd frontend
    npm install
    ```

4. Step 4:

    ```bash
    cd ../backend
    npm install
    ```

1. Step 5:
    ```bash
    cd backend
    npm start
    ```

2. Step 6:
    ```bash
    cd ../frontend
    ng serve
    ```
